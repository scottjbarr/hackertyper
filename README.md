# Hacker Typer

Output like a hacker in your shell.

Takes input from stdin.

## Usage

Defaults to 1ms delay per character.

```
cat somefile.txt | hackertyper
```

or specify a custom delay in milliseconds.

```
$ cat somefile.txt | hackertyper -delay 5
```
